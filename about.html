About CS50 Final Project

Letter-Number Converter

Project Idea

This project is a web application that converts English letters to their corresponding numbers (1-26) and vice versa. It was developed as part of the final project for Harvard University's CS50 course.

Technologies Used

Python Flask

Backend and server

HTML5

Page structure

CSS3

Styling and design

JavaScript

Interactive interface

SQLite

Data storage

Features

  • Real-time conversion without page reload
  • Multiple separator options (/, -, ., |, space)
  • Copy to clipboard functionality
  • Text-to-speech output
  • Conversion history tracking
  • Responsive design for all devices
